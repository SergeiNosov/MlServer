Запуск из командной строки (на всех ОС) - через dotnet:

dotnet Pullenti.Server.dll

Сервер запускается и взаимодействует с Pullenti SDK через http-протокол, напрямую обращаться
к серверу не надо, а использовать функцию Processor.ProcessOnServer, которая сделает всё, что надо.
По умолчанию, адрес для доступа с локальной машины http://localhost:1111, но в принципе сервер можно 
запускать и на любой машине в сети.
Для остановки сервера нужно нажать в консоли любую клавишу или прервать процесс системным образом.

Возможные параметры:
-port ... - порт, по умолчанию 1111;
-address ... - внешний адрес, по умолчанию все;
-noterminal true - не выводить никаких сообщений в консоль и не ждать нажатие клавиши для останова;
